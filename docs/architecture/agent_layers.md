## 3. Agent Communication Patterns

### 3.1 Hierarchical Collaboration Pattern

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        HIERARCHICAL COLLABORATION                            │
│                                                                             │
│                         ┌─────────────────┐                                 │
│                         │   Orchestrator  │                                 │
│                         └────────┬────────┘                                 │
│                                  │                                          │
│         ┌────────────────────────┼────────────────────────┐                │
│         │                        │                        │                │
│  ┌──────▼─────┐           ┌──────▼─────┐           ┌──────▼─────┐          │
│  │ Domain Lead │           │ Domain Lead │           │ Domain Lead │          │
│  └──────┬─────┘           └──────┬─────┘           └──────┬─────┘          │
│         │                        │                        │                │
│    ┌────┼────┐              ┌────┼────┐              ┌────┼────┐          │
│    │    │    │              │    │    │              │    │    │          │
│┌───▼─┐┌─▼─┐┌─▼─┐        ┌───▼─┐┌─▼─┐┌─▼─┐        ┌───▼─┐┌─▼─┐┌─▼─┐        │
││Agent││Agt││Agt│        │Agent││Agt││Agt│        │Agent││Agt││Agt│        │
│└─────┘└───┘└───┘        └─────┘└───┘└───┘        └─────┘└───┘└───┘        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Communication Flow:
1. Orchestrator issues high-level directives to Domain Leads
2. Domain Leads translate directives into domain-specific tasks
3. Domain Leads assign tasks to appropriate agents
4. Agents report progress and results back to Domain Leads
5. Domain Leads aggregate results and report to Orchestrator
6. Orchestrator coordinates across domains
```
